h2. Setup

h3. Server configuration

You'll need to send Steve Crouch (s.crouch@software.ac.uk):

* A copy of either your endpoint's public certificate, or a copy of the public CA certificate that signed it.
* Send either an EPR or the URI of the endpoint (depending on what your implementation supports).
* Send any username/password if you're using UsernameToken authentication at the message level.

For server configuration:

* Install the demo client certificate's CA certificate attachment:"fb4b1346.0" in your endpoint's trust store.
* Optionally set your server to mutual client authentication.

h3. Application

The Minem application (attachment:"Minem-dynamically.tar.gz" will need to be downloaded and unpacked, and the update_file program installed in a well known location, sending the installed path location to Steve Crouch (s.crouch@software.ac.uk).

When run e.g. ./update_file you should see the following:

<pre>
update_file v1.2 27-May-2004

Evaluate energy function and do quick monpo minimization.

Usage: eval infile outfile

STOP
</pre>
