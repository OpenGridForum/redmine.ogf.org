Including a JSON Schema with the JSON Rendering Specification

Added by Zdenek Sustr over 7 years ago

I have developed a JSON schema based on the latest version of the OCCI JSON Rendering specification. There is a GitHub project to maintain the schema: https://github.com/sustr4/OCCI-schema and the main schema document is currently available from https://raw.githubusercontent.com/sustr4/OCCI-schema/master/schema/OCCI-schema.json

There are currently open issues mapping to comments already mande in this forum during the public review process. Once settled, the schema will be ready for inclusion in the specification.

Opinions, bug reports and contributions are welcome.
