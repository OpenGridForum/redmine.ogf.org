Abstract:

The Network Services Framework describes a framework to support the request and management of Network Services that use the Network Service Interface (NSI) protocol. The NSI is a web-service based API that operates between a requester software agent and a provider software agent. The full suite of NSI services allows an application or network provider to request and manage network service instances. These services include the Connection Service and the Document Distribution Service.
