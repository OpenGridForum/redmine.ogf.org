DRMAA v1 Java Binding

Draft Document
08/30/2013

This is the DRMAA v1 language binding for the Java programming language, based on GFD.130.
It is widely implemented, but not formally standardized. Most implementations use the GridEngine or the GridWay implementation as reference.

Please note that DRMAA v1 is deprecated.
