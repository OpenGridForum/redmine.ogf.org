DRMAA v1 C Binding

Draft Document
08/30/2013

This is the DRMAA v1 language binding for the C programming language, based on GFD.22. It is widely implemented, but not formally standardized. Most implementations use the GridEngine header file as reference.

Please note that DRMAA v1 is deprecated.
We recommend to use the DRMAA v2 C binding instead.
