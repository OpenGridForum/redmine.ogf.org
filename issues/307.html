Description

Hello Richard,

This document describes a set of standard error codes for the NSI CS protocol. Each error code is formaly defined to ensure consistent error reporting through detailed error explanations and example XML elements.

We have separated this out of the NSI CS document since we expect that the error codes will need to be regularly updated. We would like to do this without having to update the whole NSI CS protocol.

Guy
