Description

The Network Service Interface (NSI) is a web-service based API that operates between a requester software agent and a provider software agent. The NSI incorporates a suite of services which share a common framework described in GFD.173 'Network Services Framework'.

The 'NSI Connection Service Protocol' v2.0 recommendation describes the protocol, state machine, architecture and associated processes and environment in which software agents interact to deliver a Connection. A Connection is a point-to-point network circuit that can transit multiple networks belonging to different providers.
