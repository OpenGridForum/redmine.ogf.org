Description

This is an informal session where a DRMAA2 programming infrastructure
is provided to the participants so that the DRMAA2 C API can be inspected
and discussed. This is particular useful also for participants of
the DRMAA2 language binding session.

Programming environment will consist of
- Univa Grid Engine 8.2 demo version
- DRMAA2 C API beta version

Requirements for the praticipants:
- Laptop with MacOS X, Linux or Windows (64-bit)
- Free Virtual Box
- Free Vagrant
- git tools (not required)

Can only be held on Monday or Tuesday. Probably best before DRMAA2
language binding sessions.
