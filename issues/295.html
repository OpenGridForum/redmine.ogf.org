Description

The DRMAA Version 2 root specification was published in January 2012 as GFD-R-P.194. In September 2012, a first errata was published for the root specification.

DRMAA2 is now fully adopted in Univa GridEngine. This industry-grade implementation showed some issues with the C binding, which partly also influence the root specification. Furthermore, a large set of clarifications needed to be done in the document text. Another reason for changes were the ongoing Python and OCCI binding developments.

Given the necessary changes, we created the attached updated version of the root spec.

Please check the annex chapter of the attached file for a list of changes. We leave it to the GFSG if these changes demand a new GFD number, or if they can be treated as another errata publication.
