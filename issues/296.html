Description

The DRMAA Version 2 C language binding was published in November 2012 as GFD-R-P.198.

DRMAA2 is now fully adopted in Univa GridEngine. This industry-grade implementation showed some issues with the C binding specification. Furthermore, a large set of clarifications needed to be done in the document text. Given the necessary changes, we created the attached updated version of the spec.

Please check the annex chapter of the attached file for a list of changes. We leave it to the GFSG if these changes demand a new GFD number, or if they can be treated as errata publication.
